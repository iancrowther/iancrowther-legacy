about infinitefocus

The goal for infinitefocus is to bridge the gap between the enterprise space and the world of open source. The aim through open source projects is to create viable enterprise solutions which can be incorported into new and existing architectures. web standards, web usability and web accessibility are key to our values as we aim to build a more connected web.

infinitefocus can help

We are offer a number of services, which are aimed at helping you deliver the right solution for your project or business. If you are interested by any of our current projects and would like to either pitch in or use our existing code base then please feel free to get in touch!.
